Instalação do Pytube: pip install pytube

Documentação: https://pytube.io/en/latest/

Python: https://www.python.org
--> Versão Usada: 3.13.0

Caso o pytube não funcione, use o pytubefix.
Ao rodar o código no terminal e aparecer o erro: ModuleNotFoundError: No module named 'pytubefix', verifique se o python está instalado na versão acima ou mais recente.
Verifique também a versão do pip install, se for necessário atualize.
